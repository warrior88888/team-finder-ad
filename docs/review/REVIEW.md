# TeamFinder — Вариант 1

Уважаемый ревьюер, рад приветствовать Вас на странице моего репозитория.

## Отступления от ТЗ

Поле `phone` на уровне модели `User` сделано необязательным. Поле не является обязательным при регистрации, поэтому я
посчитал нужным дать пользователю возможность удалить номер телефона при редактировании профиля.

## Доработки сверх ТЗ

Все хардкод-пути в шаблонах заменены на ссылки в Django-стиле. Исправлен CSS для корректного отображения аномально
длинных строк — ранее сетка ломалась. Доработан JS: счётчик участников проекта теперь не уходит в отрицательные
значения. Также введён дополнительный функционал по безопасности, админке, метрикам и развёртыванию на
продакшен-сервере.

## Доработки после ревью

Спасибо за ревью! Я привел проект по всем правкам, справился с магическими числами путем введения констант. Полностью
исправлены urls, никакого абстрактного pk - только project_id и user_id. За время редактирования всех файлов шаблонов я
понял важность возможности быстро изменять всю логику, когда в коде нет хардкода. Вместе с константами я ввел
get_absolute_url в модели, привел их в порядок. Убрал .js файлы в static/ для корректного добавления хэша, а также более
чистой структуры. Буду использовать константы в дальнейшем и избегать хардкода.
