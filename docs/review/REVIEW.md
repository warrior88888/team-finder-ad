# TeamFinder — Вариант 1

Уважаемый ревьюер, рад приветствовать Вас на странице моего репозитория.

## Отступления от ТЗ

Поле `phone` на уровне модели `User` сделано необязательным. Поле не является обязательным при регистрации, поэтому я посчитал нужным дать пользователю возможность удалить номер телефона при редактировании профиля.

## Доработки сверх ТЗ

Все хардкод-пути в шаблонах заменены на ссылки в Django-стиле. Исправлен CSS для корректного отображения аномально длинных строк — ранее сетка ломалась. Доработан JS: счётчик участников проекта теперь не уходит в отрицательные значения. Также введён дополнительный функционал по безопасности, админке, метрикам и развёртыванию на продакшен-сервере.

## Тестовые данные

Все пользователи с почтой на домене `@example.com` имеют пароль `devpass` — будет исправлено после успешной сдачи.
